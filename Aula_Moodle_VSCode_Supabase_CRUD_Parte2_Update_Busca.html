CONTEÚDO + PRÁTICA
1) Update no formulário (modo “editar”)

Ideia principal

Para editar, precisamos de 3 coisas:

  1. Um botão Editar em cada item da lista.
  2. Ao clicar, carregar os dados desse aluno nos inputs (nome/email/curso).
  3. Ao salvar, executar update() com filtro eq('id', idDoAluno) para não alterar todo mundo.

Trecho essencial (App.jsx): estados + função editar + salvar

Copie e adapte no seu App.jsx. (Você vai encaixar no seu código atual.)

// 1) novo estado para controlar edição
const [editandoId, setEditandoId] = useState(null);

// 2) quando clicar em Editar, carregue os dados no formulário
function editarAluno(aluno) {
  setEditandoId(aluno.id);
  setNome(aluno.nome ?? '');
  setEmail(aluno.email ?? '');
  setCurso(aluno.curso ?? '');
}

// 3) uma função "salvar" que decide: INSERT ou UPDATE
async function salvarAluno(event) {
  event.preventDefault();

  // validação simples
  if (nome.trim() === '') {
    alert('O nome é obrigatório.');
    return;
  }

  if (editandoId === null) {
    // CREATE (insert)
    const { error } = await supabase
      .from('alunos')
      .insert([{ nome, email, curso }]);

    if (error) { console.error(error.message); return; }
  } else {
    // UPDATE (update) - SEMPRE COM FILTRO
    const { error } = await supabase
      .from('alunos')
      .update({ nome, email, curso })
      .eq('id', editandoId);

    if (error) { console.error(error.message); return; }
  }

  // limpa o formulário e sai do modo edição
  setNome(''); setEmail(''); setCurso('');
  setEditandoId(null);

  // atualiza a lista na tela
  listarAlunos();
}
Dica: quando editandoId for diferente de null, seu botão pode trocar de “Cadastrar” para “Salvar alterações”.

    2) Busca por curso + ordenação por nome

    Como funciona a busca

    A busca (filtro) é só uma condição a mais no select. Exemplo: se o aluno digitar “Info”, a aplicação lista apenas os alunos cujo curso contenha “Info”.

    Trecho (App.jsx): filtroCurso + listarAlunos com filtro

    const [filtroCurso, setFiltroCurso] = useState('');

async function listarAlunos() {
  let query = supabase
    .from('alunos')
    .select('*')
    .order('nome', { ascending: true });

  // se tiver filtro, aplica (contém / ignorando maiúsculas e minúsculas)
  if (filtroCurso.trim() !== '') {
    query = query.ilike('curso', '%' + filtroCurso.trim() + '%');
  }

  const { data, error } = await query;

  if (error) {
    console.error(error.message);
    return;
  }

  setAlunos(data);
}
    Dica: order('nome') deixa a lista organizada e facilita encontrar um aluno.

    No Supabase (SQL Editor)

    Para evoluir a tabela, adicione uma nova coluna:

    alter table alunos
add column telefone text;
